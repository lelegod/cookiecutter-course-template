# Cookiecutter Course Template

This is a template for DTU course by Kyle Nathan Yahya.

## Overview

This template provides a structured starting point for DTU course work, featuring:

- **Automated Tasks**: Uses `invoke` for common tasks like environment setup.
- **VS Code Configuration**: Pre-configured settings for a smooth development experience.
- **LaTeX Integration**: Setup for writing course notes in LaTeX.
- **Python Support**: Ready-to-use Python environment setup.

## Usage

To use this template to start a new project:

1.  Install cookiecutter:
    ```bash
    pip install cookiecutter
    ```

2.  Run cookiecutter with this template:
    ```bash
    cookiecutter .
    ```
